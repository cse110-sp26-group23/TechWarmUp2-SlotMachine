### Feature 1 — Symbol Images in Reels and Paytable
As a player,
I want the slot symbols to appear as proper icons instead of emojis 
So the game can feel more polished and visually stronger.
 
Acceptance Criteria:
- Each slot symbol should be displayed using a PNG image on both the reels and the paytable
- If a symbol image cannot be loaded, the system automatically shows as an emoji fallback.
- Any failed image element is removed from the DOM, so the image icon never appears.
