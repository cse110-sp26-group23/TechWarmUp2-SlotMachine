### Feature 1 — Basketball Theme
As a player,
I want the symbols to match a basketball theme,
so that the game feels cohesive and fun instead of generic.
 
Acceptance Criteria:
- All 7 symbols are basketball-themed (Basketball, Sneaker, Jersey, Trophy, All-Star, Hot Streak, Championship Ring)
- Color palette uses NBA orange and arena dark tones
- Emoji fallback is available if symbol images fail to load
