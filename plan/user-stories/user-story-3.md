### Feature 3 — Real Sound Effects
As a player,
I want actual audio files for spin, stop, and win sounds,
so that the game sounds like a real slot machine and not a browser beep.
 
Acceptance Criteria:
- MP3 files are used for spin, stop tick, small win, big win, and jackpot sounds
- If an audio file is missing, the game continues silently without errors
- Mute toggle silences all sounds including background music immediately
