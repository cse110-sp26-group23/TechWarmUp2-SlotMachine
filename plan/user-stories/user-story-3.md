### Feature 3 — Real Sound Effects
As a player,
I want the game to use audio effects for the spinning and wins,
so that it feels like a real slot machine instead of just having the browser beep after everything.
 
Acceptance Criteria:
- The game uses an MP3 audio file for the spinning, small wins, big wins, stopping, and jackpot sounds.
- If any audio file fails to load, the game will continue running without giving any error and will simply play no sound.
- A mute toggle will be added to instantly disable the audio, including sound effects and background music
