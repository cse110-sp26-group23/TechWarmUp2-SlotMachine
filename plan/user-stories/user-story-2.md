### Feature 2 — Scrolling Reel Animation
As a player,
I want the slot reels to spin and be smooth with the symbols,
so that each spin can feel lively instead of the symbols appearing instantly.
 
Acceptance Criteria:
- When a spin starts, each scroll should be in sequence with the random symbol before showing the results
- The reel motion shows gradually without easing the curve as it approaches the symbol
- Reels stop after another from left going right, creating a staggered effect.
