# Slot Machine Visuals

Slot machine visuals are one of the main things that shape the player’s first impression of the game. For our project, visual research helps us decide what kind of slot machine experience we want to create and what reference points we should give to Claude Code when prompting for UI ideas.

## Why Visual Design Matters

- [ ] **First impression**
  - The visuals make the game feel polished, fun, cheap, chaotic, retro, or modern.
  - A good visual design can make even a simple game feel much more complete.

- [ ] **Readability**
  - Players need to understand what they are seeing.
  - Buttons, reels, credits, payouts, and messages should all be easy to read.

- [ ] **Feedback**
  - Visual effects help communicate when the reels are spinning, when a result is revealed, and when the player wins.
  - Animation can make the experience feel more rewarding, but it should not hide important information.

- [ ] **Theme / identity**
  - Slot machines often use themes like fruit, casino classic, neon, treasure, lucky 7s, animals, fantasy, or sci-fi.
  - Even if our project is not a real casino product, having a clear theme can make it feel more intentional.

## Common Visual Elements in Slot Machines

### Symbols
Symbols are the visual pieces shown on the reels. Classic examples include:
- cherries
- lemons
- bars
- bells
- lucky 7s
- diamonds
- stars

More modern slot machines often use themed symbols tied to the game’s setting.

### Paytable
A paytable explains which symbol combinations win and how much they pay. This is important visually because it helps the player understand the rules instead of guessing.

### Credit / Bet / Win Display
Most slot interfaces include clearly visible counters for:
- current credits
- current bet
- payout from the last spin
- sometimes total win or balance change

### Buttons
Common controls include:
- spin
- increase/decrease bet
- mute or sound toggle
- help or rules
- sometimes reset

These buttons should stand out visually and be easy to identify.

## Visual Style Ideas for Our Project

Here are a few possible directions our project could take:

- [ ] **Classic casino style**
  - red, gold, black
  - shiny buttons
  - cherries, 7s, bells, bars
  - traditional slot machine look

- [ ] **Arcade / fun style**
  - bright colors
  - exaggerated animations
  - playful symbols
  - more humorous tone

- [ ] **Minimal / clean style**
  - fewer visual distractions
  - simple reel design
  - modern flat UI
  - easier to test and easier to keep readable


## Important Visual Design Takeaways

- [ ] **Do not overload the screen**
  - Too many flashing elements, colors, or effects can make the game look messy.

- [ ] **Make important information visible**
  - Credits, bet amount, and payouts should be easy to find at all times.

- [ ] **Use animation to support clarity**
  - Animation should make the spin feel exciting, but it should not confuse the player about what actually happened.

- [ ] **Design for responsiveness**
  - The layout should still make sense on smaller screens.

- [ ] **Match the visuals with the sound and math**
  - If the game is meant to feel casual and fun, the visuals should match that tone.
  - If the game uses rare big wins, the visual effects for those wins should feel special.

## What This Means for AI Prompting

This research gives us useful ideas to include when prompting Claude Code. Instead of asking for a generic slot machine UI, we can be more specific, such as:

- use a classic casino color palette
- include visible credit, bet, and win counters
- make the spin button prominent
- add a readable paytable section
- use simple but clear spin and win animations
- keep the layout responsive and not overly cluttered

The more specific we are about visuals, the more likely the generated result will match what we actually want.
